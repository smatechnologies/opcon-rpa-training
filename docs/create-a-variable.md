## Exercise: Create a User Variable

### Objective 

Create a User Variable in the OpCon RPA Client

### Instructions 

1. In the RPA Tray Client, click to create a New Robot Task.
2. Click  the **Variables** button in the bottom left-hand corner.
3. Click **Add User Variable**.
4. Name the variable **rpatestvar**.
5. Leave the **Variable value** field blank for the initial value.
6. Click **Add**.
7. Notice the syntax used for variables in the RPA system `{USERVAR(rpatestvar)}`.
8. Click **Close**.
9. Click **Cancel** to close this Robot task without saving (The user variable will remain saved).
