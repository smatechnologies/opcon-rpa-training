## Create an OpCon RPA Workflow Process

## Objective

Create a workflow combing all three aspects of the RPA Orchestration process covered in this training.

## Summary

Orchestrate an OpCon RPA Web Macro Task with an OpCon Job through a Self Service Button.

- Create a robot task, build an OpCon RPA Job, and a Self Service button to run the Job

    - Your workflow should follow this pattern: **Self Service Button --> OpCon Schedule and Job --> RPA Task**

## Instructions

- Create an RPA Web Macro task that collects a value from a website and stores it in a User Variable OR downloads a file to a local destination. 

- Create and build an OpCon Schedule and Job to run the Web Macro task. 

- Create a Self Service button that will kick off a run of the OpCon Job/Web Macro task.

- Validate your workflow and verify your work with the instructor.
