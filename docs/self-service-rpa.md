# Using Self Service for OpCon RPA Jobs

Creating a Self Service button in OpCon's Solution Manager allows users to RPA tasks on demand.

Example: Instead of emailing IT to update rates on a website, a user clicks a button labeled “Update Rates” in Self Service. OpCon runs the automation behind the scenes.

Using Self Service for RPA provides:

- Faster turnaround for repetitive tasks
- Eliminates manual errors
- Provides audit train for compliance
- Maintains control through Role-based access control
- Centralized monitoring and alerted
